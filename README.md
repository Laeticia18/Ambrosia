# Ambrosia Restaurant Website

## Project Overview
Ambrosia is a responsive website designed for a restaurant. It showcases the restaurant's menu, location, hours, and special events. This project aims to provide an excellent user experience and easy navigation.

## Features
- **Menu Display**: Detailed menu with images and descriptions.
- **Location Information**: GPS directions and a map to help customers find the restaurant.
- **Contact Form**: Allows customers to make reservations or inquiries.
- **Responsive Design**: Optimized for both mobile and desktop users.

## Technologies Used
- **HTML5**: Structure of the website.
- **CSS3**: Styling and layout.
- **JavaScript**: Interactivity and dynamic content.
- **React.js**: Frontend framework for building user interfaces.
- **Node.js**: Backend server handling requests and serving the front end.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Laeticia18/Ambrosia.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Ambrosia
   ```
3. Install the required packages:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## Usage
- After starting the server, open your browser and navigate to `http://localhost:3000` to view the website.
